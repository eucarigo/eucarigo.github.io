PreCita v0.36.8

Gestión de citas y recordatorios automáticos

Aplicación de escritorio para Windows desarrollada en Python que permite gestionar citas y contactos, sincronizar eventos con Google Calendar y enviar recordatorios por correo electrónico a través de Gmail.

¿Para quién es PreCita?

Está pensada para profesionales que trabajan con citas programadas (clientes o pacientes), como:

  • Psicólogos
  • Médicos
  • Fisioterapeutas
  • Esteticistas
  • Masajistas
  • Entrenadores personales

En definitiva, para cualquier profesional con una agenda ocupada que requiera automatizar el envío de recordatorios.

Características principales

  • Sincronización directa con Google Calendar en ventana configurable de 7 a 49 días (15 por defecto)
  • Recordatorios automatizados y personalizables por email mediante Gmail
  • Vistas de agenda: diaria, semanal y mensual, con navegación por períodos y acceso rápido a "Hoy"
  • Gestión de contactos: alta, edición, eliminación y vinculación con citas
  • Plantilla de correo editable con asunto y cuerpo en HTML, formato básico (negrita/cursiva/subrayado/listas) y variables dinámicas
  • Adjuntos reutilizables en plantilla con controles de seguridad y límite de tamaño total (16 MB)
  • Completamente local y privado, sin telemetrías
  • Encriptación opcional de base de datos mediante contraseña
  • Software libre distribuido bajo GNU GPL v3

Tecnologías

  • Python 3.10+ (3.12 recomendado)
  • PyQt6
  • SQLite
  • cryptography
  • Google Calendar API
  • Gmail API

Instalación y ejecución

Requisitos previos:

  • Python 3.10 o superior instalado. Se recomienda Python 3.12.
  • Sistema Operativo Windows 10 o superior. Se recomienda Windows 11.

1. Clonar o descargar el repositorio

git clone https://github.com/eucarigo/precita.git
cd precita

2. Crear y activar entorno virtual

python -m venv venv
venv\Scripts\activate

3. Instalar dependencias

pip install PyQt6 PyQt6-WebEngine google-auth google-auth-oauthlib google-api-python-client pytz cryptography

4. Ejecutar la aplicación

python main.py

Configuración y almacenamiento local

PreCita guarda su información en la carpeta local del usuario ~/.precita/:

  • precita.db: base de datos SQLite
  • db_encryption_config.json: estado local de encriptación de la base de datos
  • token.json: token OAuth local para Google
  • client_secret.json: opcional, para flujo OAuth alternativo
  • template_attachments/: copia local segura de adjuntos de plantilla

Desde el menú principal puede abrir Datos locales para consultar esta carpeta y, cuando lo necesite, ejecutar Optimizar (comando SQLite VACUUM) para compactar precita.db.

Encriptación de base de datos

En Datos locales puede abrir Encriptar base de datos para:

  • Elegir entre "Deshabilitar encriptación (predeterminado)" y "Habilitar encriptación (recomendable)"
  • Definir una contraseña alfanumérica obligatoria para proteger precita.db
  • Introducir la contraseña al arrancar PreCita cuando la base de datos está cifrada

Nota de seguridad: Se usa la variable de entorno PRECITA_MASTER_KEY para desencriptar el fichero config.bin que contiene las claves de la API de Google.

Flujo de uso recomendado

  1. Abra PreCita
  2. Use Sincronizar para iniciar sesión con Google (si aplica) y traer citas del calendario seleccionado
  3. Revise o cree contactos en Gestión de contactos o Añadir contacto
  4. Ajuste asunto/cuerpo/adjuntos desde Personalizar plantilla
  5. Revise Datos locales para verificar el estado de ~/.precita/, usar Optimizar y configurar Encriptar base de datos si desea proteger precita.db con clave
  6. Ejecute Lanzar pendientes para envío manual o deje activo el envío automático

Variables disponibles en la plantilla

Puede usar estas variables en el asunto y cuerpo de los correos de recordatorio:

  • {nombre_citado}
  • {apellidos_citado}
  • {correo_citado}
  • {tlf_citado}
  • {hora_cita}
  • {fecha_cita}
  • {dia_semana}

Reglas de seguridad para adjuntos

  • Se bloquean extensiones ejecutables o de riesgo según política de Gmail
  • Se permite *.zip solo si su contenido no incluye archivos bloqueados
  • Se bloquean otros formatos comprimidos (.rar, .7z, .tar, .gz, .bz2, .xz)
  • El tamaño combinado de cuerpo + adjuntos está limitado a 16 MB

Configuración funcional

Desde Configuración (también con Ctrl+,):

  • Tema visual (Claro / Oscuro)
  • Intervalo de revisión automática de recordatorios
  • Días de sincronización de calendario (rango 7-49)
  • Calendario de Google seleccionado
  • Activación/desactivación de notificaciones de Windows
  • Inicio con Windows
  • Desvincular cuenta de Google en este equipo
  • Restablecer valores predeterminados

Desde la ventan Ayuda (también con Ctrl+H) puede abrir enlaces directos a funciones internas como sincronización, contactos, plantilla, configuración y almacenamiento.

Otros atajos de teclado

  • Ctrl+H: Abrir la ventana Ayuda
  • Ctrl+,: Abrir la ventan Configuración
  • Ctrl++: Aumentar tamaño de la app (máximo 150%)
  • Ctrl+-: Disminuir tamaño de la app (mínimo 80%)

Política de privacidad

PreCita es una herramienta de ejecución local y prioriza el control del usuario sobre sus datos:

  1. Datos locales: contactos, citas y configuración se guardan en el equipo del usuario
  2. Uso de APIs de Google: se usa Google Calendar para lectura de eventos y Gmail para envío de recordatorios
  3. Transferencia de datos: no hay telemetría ni envío a terceros fuera de las APIs necesarias de Google
  4. Control y eliminación: para borrar datos, cierre la app y elimine ~/.precita/; opcionalmente revoque el acceso en su cuenta de Google
  5. Licencia: software distribuido bajo GNU GPL v3, sin garantías. Puede ver el fichero LICENSE al descargar la aplicación o entrar en el enlace oficial

Estructura del proyecto

  • main.py: interfaz, lógica de negocio, persistencia, sincronización y envío
  • VERSION: versión de aplicación
  • precita.ico: icono principal
  • LICENSE: licencia GNU GPL v3
  • config.bin: binario cifrado con las claves de la API de Google
