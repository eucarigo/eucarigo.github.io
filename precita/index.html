PreCita

Gestor de Citas y Contactos Inteligente

Aplicación de escritorio en Python para gestionar tus citas y contactos, con sincronización automática de Google Calendar y recordatorios por correo electrónico.

Ver en GitHub Descargar

Características Principales

Sincronización con Google Calendar

Sincroniza automáticamente tus citas con tu Google Calendar para tener todo organizado en un solo lugar.

Recordatorios por Email

Recibe recordatorios automáticos por correo electrónico antes de tus citas importantes.

Gestión de Contactos

Almacena y organiza información detallada de tus contactos para un acceso rápido.

Notificaciones Inteligentes

Sistema de notificaciones configurable para nunca olvidar una cita importante.

Estadísticas y Reportes

Visualiza estadísticas sobre tus citas y genera reportes para un mejor seguimiento.

Base de Datos Local

Almacenamiento seguro de tus datos localmente con opción de backup automático.

Instalación

Requisitos del Sistema

Mínimos:

  • Python 3.8 o superior
  • 4 GB de RAM
  • 500 MB de espacio en disco
  • Conexión a internet para sincronización

Recomendados:

  • Python 3.12
  • 8 GB de RAM
  • 1 GB de espacio en disco
  • Conexión estable a internet

Pasos de Instalación

1

Clonar el repositorio

git clone https://github.com/eucarigo/precita.git
cd precita
2

Crear entorno virtual

python -m venv venv
source venv/bin/activate # Linux/Mac
venv\Scripts\activate # Windows
3

Instalar dependencias

pip install -r requirements.txt
4

Configurar credenciales

Crea un archivo .env con tus credenciales de Google:

GOOGLE_CALENDAR_ID=tu_calendar_id@gmail.com
GMAIL_USER=tu_email@gmail.com
GMAIL_PASSWORD=tu_password_app
5

Ejecutar la aplicación

python main.py

Guía de Uso

Crear una Cita

  1. 1. Haz clic en "Nueva Cita"
  2. 2. Completa los detalles (título, fecha, hora)
  3. 3. Selecciona los contactos involucrados
  4. 4. Configura recordatorios si es necesario
  5. 5. Guarda y sincroniza con Google Calendar

Gestionar Contactos

  1. 1. Ve a la sección "Contactos"
  2. 2. Haz clic en "Agregar Contacto"
  3. 3. Ingresa información personal y de contacto
  4. 4. Asocia etiquetas para mejor organización
  5. 5. Guarda para usar en futuras citas

Sincronización

  1. 1. Configura tu cuenta de Google
  2. 2. Autoriza el acceso a Calendar y Gmail
  3. 3. Selecciona el calendario a sincronizar
  4. 4. Activa la sincronización automática
  5. 5. Revisa el estado de sincronización

Configurar Recordatorios

  1. 1. Accede a "Configuración"
  2. 2. Ve a "Recordatorios"
  3. 3. Configura tiempos por defecto
  4. 4. Personaliza mensajes de email
  5. 5. Activa notificaciones del sistema
