Documentación de PreCita

Características

  • Integración sencilla con diversas plataformas
  • Interfaz de usuario intuitiva
  • Capacidades sólidas de gestión de datos
  • Informes y análisis completos

Instrucciones de instalación

Para instalar PreCita, sigue estos pasos:

  1. Descarga la última versión desde este enlace.
  2. Descomprime el paquete.
  3. Ejecuta el script de instalación mediante install.sh.
  4. Sigue las instrucciones en pantalla para completar la instalación.

Tutorial

Empieza a usar PreCita:

  1. Abre la aplicación y crea un nuevo proyecto.
  2. Añade tus fuentes de datos y configura los ajustes.
  3. Utiliza el panel para ver información y análisis.
  4. Exporta informes cuando lo necesites.

Preguntas frecuentes

P: ¿Es PreCita gratuito?

R: Sí, PreCita es de código abierto y se puede usar de forma gratuita.

P: ¿Dónde puedo encontrar soporte?

R: Puedes encontrar soporte en nuestros foros oficiales y en la página de incidencias de GitHub.
